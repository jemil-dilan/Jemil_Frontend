# JEMIL Design System - African-Inspired Bus Booking Platform

## 🎨 Color Palette

### Primary (African Earth & Sunsets)
- **Terracotta:** `#D2691E` - Main brand, CTAs
- **Burnt Orange:** `#CC5500` - Hover states
- **Ochre Gold:** `#CC7722` - Highlights

### Secondary (Warmth & Trust)
- **Burgundy:** `#800020` - Premium sections
- **Desert Sand:** `#EDC9AF` - Light backgrounds
- **Rich Brown:** `#654321` - Text

### Accent (Energy)
- **Coral:** `#FF6B6B` - Notifications
- **Golden:** `#FFD700` - Premium badges

## ✍️ Typography

```
Headlines:  Montserrat (800 weight) - Bold, geometric
Body:       Merriweather (400 weight) - Readable, warm
Accent:     Raleway (300-700 weight) - Elegant
```

## 🎭 Key Animations

- **float:** 3D bus floating motion
- **slideInUp:** Hero content entrance
- **pulse:** Live availability indicators
- **patternBurst:** Booking success celebration

## 📦 Core Components

- **Header** - Navigation with mobile menu
- **BookingWidget** - Hero booking form
- **RouteCard** - Popular route display
- **ThreeBackground** - 3D particle system
- **Footer** - Links & payment methods

## 🌍 African Design Patterns

- Diagonal stripes (textile-inspired)
- Curved section dividers (landscape-like)
- Warm color gradients (sunset palette)
- Geometric patterns (subtle overlays)

## ♿ Accessibility

- WCAG 2.1 AA compliant
- 4.5:1 contrast ratios
- 44px touch targets (mobile)
- Keyboard navigation support
- Screen reader friendly

## 📱 Responsive

- Mobile-first (375px base)
- Breakpoints: 768px, 1280px, 1920px
- Touch-friendly buttons
- Simplified mobile booking flow
